Bookmarklets

October 27, 2022

What are bookmarklets?

They are small snippets of JavaScript code that are in the form of a bookmark. Since it is basically a JavaScript program, it is natively available in all major browsers, including Mozilla Firefox & Chromium-based browsers.

We’ll look at how to create bookmarklets in Firefox.

How to Create Bookmarklets

In Mozilla Firefox, open your bookmarks toolbar by pressing CTRL + B, you can then right-click

and click on “Add Bookmark…”

We can write a JS script in the URL section of the prompted dialog box.

javascript:(()%3D%3E%7Balert(%27Hello%2C%20World!%27)%3B%7D)()%3B

Give a name to the bookmarklet and click on Save.
What this bookmarklet is going to do it open a alert window in the browser with the message “Hello World”.

All bookmarklets after the “javascript:” part have to be url encoded.

Similarly, we can make another bookmarklet that would replace the domain in the URL with whatever we want.

javascript:window.open(location.href.replace(%2F%3A%5C%2F%5C%2F(%5B%5Cw-%5D%2B.)%3F(reddit%5C.com)%5C%2F%2Fi%2C%20%22%3A%2F%2Fl.opnxng.com%2F%22)%2C%20%22_self%22)

This will replace the host name “reddit.com” with an alternative front-end “l.opnxng.com”

There are many other bookmarklets that do a variety of things:

  • agentcooper’s youtube playlist sort - which will sort a given youtube playlist by the number of views of the videos in it.
  • Twitter keyword + Date search - using this we can search for a keyword on Twitter within the given time period.
